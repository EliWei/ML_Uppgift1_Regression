# ML enskild uppgift – California Housing, Spår A Regression

## Beskrivning
Regression-modell för att förutsäga medianvärdet (median_house_value) baserat på områdets egenskaper.

## Miljö
- Python 3.13

## Kör projektet
1. Skapa en virtual environment: `python -m venv .venv`
2. Aktivera: `source .venv/bin/activate`
3. Installera beroenden: `pip install -r requirements.txt`
4. Öppna notebook: `jupyter notebook ML_Uppgift1_Elisabeth_Weise.ipynb`
5. Kör: Restart & Run All

## Data
Datasetet `housing.csv` ingår inte i repot. Ladda ner det separat och lägg det i projektmappen innan du kör notebooken.
